##  Solo Workflow Note

Even though you are one person, **switch roles intentionally**:

```
  As Developer     → Focus on building features that work
  As QA            → Actively try to break your own system
  As Scrum Master  → Check if you're actually following the plan
```

> **Tip:** Time-box each "hat." Don't QA while you're still building — it splits focus and misses bugs.
